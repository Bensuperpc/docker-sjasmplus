#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
git clone https://github.com/Bensuperpc/docker-sjasmplus.git
```
```sh
cd docker-sjasmplus
```
#### Usage

```sh
./sjasmplus.sh sjasmplus hello.asm --raw=hello.rom
```
#### Build
```sh
make linux/amd64 or linux/arm/v7
```
